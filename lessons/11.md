### Plurals and Genitive

Both plurals and the genitive case (possessive case) are formed in the same way. You add one of the following to the base: -, -, -

| pattern | example | 
| --- | --- |
| - | 路ゐ 答 |
| - | 路别  |
| - | 路○ю  |

Some plurals have irregular formations. They will not change (eg. "deer") or they will change vowels (eg. "men").

### Irregular plurals

Some English nouns go through vowel changes in order to show plural number:

| pattern | example | 
| --- | --- |
|-> | ヰ->ヰю |
|-> | ヰ->ヰ拆 |
| -> | 叼->梆, 叼->梆 |
| -> |  梆|
| -> | юゐ->юゐ, ｐ叼->ｐ叼, ｐ->ｐ, ゐ答->ゐ答, 拆->拆, 梆->梆, ゐ->ゐ |
| ->梆 | 拆->拆梆, 别->别梆 |

And some nouns don't change at all when plural. For example: , 梆, 吼. Only the grammar and context will tell you that these nouns are plural or singular.

### Regular (Weak) Verbs

Regular verbs, otherwise known as "weak" verbs, form the past tense by adding one of the following to the base: -, -, -
